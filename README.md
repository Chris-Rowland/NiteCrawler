# NiteCrawler
MoonLite NiteCrawler ASCOM Focuser and Rotator Driver
This is the installer for the MoonLite NiteCrawler ASCOM Focuser and Rotator Driver. To install download and Unzip the file and run the installer .EXE file.  ASCOM version 6.2 or better is required, install this first.  Please contact me if you have problems, I will almost certainly need a log file.
